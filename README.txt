При запуске программы необходимо указать параметры
Путь к каталогу-источнику Путь к каталогу-реплике Интервал синхронизации в секундах Путь к каталогу для записи логов
Пример:
python3 /home/kirill/Task2/main.py /home/kirill/Test1/ /home/kirill/Test2/ 3 /home/kirill/
Где: /home/kirill/Task2/main.py - запуск скрипта
     /home/kirill/Test1/ - путь к каталогу-источнику
     /home/kirill/Test2/ - путь к каталогу-реплике
     3 - интервал синхронизации
     /home/kirill/ - путь к каталогу записи логов
